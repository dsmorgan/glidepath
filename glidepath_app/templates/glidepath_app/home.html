Welcome to Glidepath

Glidepath is a comprehensive investment allocation management system designed to help you manage retirement glide paths, funds, accounts, and portfolios with ease.

Rules

Import, manage, and visualize glidepath allocation rules based on retirement age bands.

Funds

Query ticker information from multiple financial data sources including yfinance, Alpha Vantage, Finnhub, and more.

Accounts

Track and manage investment accounts across different retirement vehicles.

Portfolios

Build and manage comprehensive investment portfolios with multiple accounts and funds.

Modeling

Run simulations and model different retirement scenarios and allocation strategies.

Settings

Configure API keys for financial data sources used in the Funds page.

Getting Started

Begin by importing your glidepath rules in the Rules section. You can upload CSV files containing allocation rules by retirement age bands. To query ticker information, configure your API keys in Settings and then use the Funds page to search.
