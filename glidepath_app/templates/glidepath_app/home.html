Getting Started

{% if is_admin %}

As an administrator, follow these steps to set up the system:

  1. Import glidepath allocation rules in the Rules section by uploading CSV files.
  2. Configure API keys in Settings to enable ticker information queries.
  3. Use the Funds page to search and manage fund information.
  4. Upload your account positions in the Accounts section.
  5. Create portfolios in the Portfolios section to group your accounts.
  6. Run retirement scenario simulations in the Modeling section.

The Settings page also allows you to manage user accounts, identity providers, API keys, and session settings.
